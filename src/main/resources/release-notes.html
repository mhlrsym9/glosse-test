Altered the algorithm that determines what is a phrase. The new logic says that if any of the four punctuation marks are anywhere in the string, mark it as a phrase. The old logic just looked at the first and last characters.
03/16/2011
We have split off the sound pathing algorithm into two flavors: MARSOC and non-MARSOC. For MARSOC, the content is expected to always be in alphabet/glossary so we build our paths based off of that assumption. For non-MARSOC< we determine the location of the B4X folder and use that when we calculate the path.
